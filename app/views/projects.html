Uniq

Uniq is built for prospective students looking for an easy way to explore all the programs in any school all at the same time and in a personalized way. Go into explore view and take a peak at any program, if it sparks your interest start your indepth research all within the app! Tell us about yourself, and we will recommend which programs are a good fit and if you have the marks to make programs you're interested in. Uniq currently let's you explore many programs, with many features coming soon!

Challenges

Designing a generic model for all the schools, faculties, and programs because of the large variety and fluctuations in data. We are still working on adding more properties to these models to allow for more indepth research for the users. We also created a framework to populate new records with historical data and data from other models, so data insertion will be much easier.

TDL

TDL is a to do list with many features in mind. Currently, we have hashtag grouping, by just including hashtags in your tasks we can detect and group these hashtags. We also have a custom interface which groups tasks by dates and an overhead calendar to easily view what you have to do in the short and long term. We have plans for reminders through notifications and better integration between the calendar and the tasks.

Challenges

Building the listview took time because it include a few adapters and nested listviews to allow for the many features: expandable headers, swipe-to-delete, and tap to edit.

Nibbble

Nibbble is an Android app that uses the Dribbble API to cycle through images and set them as your wallpaper. The default mode grabs random pictures from the API but if you have an account, it can select images from people you're following!

Challenges

Creating the background thread that controls the setting of the wallpaper and using the Wallpaper Service because I haven't used it before.
